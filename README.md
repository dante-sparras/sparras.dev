# Sparras.dev (Personal Website)

## Stack

- **Next.js**
- **shadcn/ui**
- **Tailwind CSS**

## Development Setup

**Clone the repository**:

```bash
git clone https://github.com/dante-sparras/sparras.dev.git
cd sparras.dev
```

**Install dependencies**:

```bash
bun install
```

**Run the development server**:

```bash
bun dev
```

The website should now be running at `http://localhost:3000`. Open this URL in your web browser to view the site.

## Available Scripts

In the project directory, you can run:

- `bun dev`: Runs the app in development mode.
- `bun build`: Builds the app for production.
- `bun start`: Starts the production server.
- `bun lint`: Runs the linter to check for code quality issues.
- `bun format`: Formats the code using Prettier.
